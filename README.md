# 🎬 Netflix Clone (HTML & CSS)

🚀 A responsive Netflix landing page clone built using only HTML and CSS.  
This project is created to practice frontend development skills and understand real-world UI layout design.

---

## 📌 Overview
This project replicates the Netflix homepage UI to improve understanding of:
- Website layout structure
- Responsive design principles
- CSS styling techniques
- Flexbox and positioning
- Real-world UI recreation

This is a frontend practice project focused on building strong fundamentals.

---

## 🛠️ Tech Stack
- HTML5  
- CSS3  

---

## ✨ Features
- 🎯 Netflix-inspired homepage design
- 📱 Responsive navigation bar
- 🎬 Hero banner section with background image
- 📺 Movie category row layout
- 🖱️ Hover effects for UI elements
- 🎨 Clean and modern interface design

## 🚀 How to Run Locally
1. Clone the repository
   git clone https://github.com/your-username/netflix-clone.git

2. Open project folder
   cd netflix-clone

3. Run project
   Open `index.html` in your browser
